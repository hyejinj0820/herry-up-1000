자주 묻는 질문
🌐 크롬 알림 허용
1크롬 주소창에 아래를 복붙해서 접속
chrome://settings/content/notifications
2이 사이트 주소가 허용됨 목록에 있는지 확인. 없으면 추가 클릭
🪟 윈도우 시스템 설정
3윈도우 설정 → 시스템 → 알림 → 목록에서 Google Chrome 찾아서 켜기
4집중 지원이 켜져있으면 알림이 막혀요 → 끄기로 변경
🔷 엣지(Edge) 사용 시
5엣지 주소창에 아래를 복붙해서 접속
edge://settings/content/notifications
6이 사이트 주소 허용됨 목록에 추가 후 위 3~4번 동일하게 진행
🌐 크롬 알림 허용
1크롬 주소창에 아래를 복붙해서 접속
chrome://settings/content/notifications
2이 사이트가 허용됨 목록에 있는지 확인. 없으면 추가 클릭
🍎 맥 시스템 설정
3시스템 설정 → 알림 → 목록에서 Google Chrome 찾아서 알림 허용 켜기
4오른쪽 상단 메뉴바에서 집중 모드가 켜져있으면 알림이 막혀요 → 꺼주세요
5그래도 안 되면 크롬 주소창에 아래 복붙 후 Enabled로 변경하고 재시작
chrome://flags/#enable-native-notifications
